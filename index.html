GitFinder

I developed this web application as a solution to one of "Frontend Mentor Challenges". It utilises the GitHub API as its core component, enabling users to initiate searches for GitHub profiles by specifying a particular username. Upon request, the application generates a summary of the searched user's profile, emphasizing key information such as public repositories, preferred programming languages, and user engagement metrics on the GitHub platform. The implementation of this application leverages a combination of web technologies, including HTML, CSS, JavaScript, and integration with the GitHub API.

Stationery Management System

I have developed an advanced client-server Java application as a solution for one of my Applications Development milestone projects at Cape Peninsula University of Technology. This application was developed using the Java programming language, Apache Derby database and the NetBeans Integrated Development Environment (IDE). It is designed to provide a efficient platform catering to the needs of both stationery users and administrators. Its principal aim is to enhance and optimize the processes involved in tracking, procuring, and distributing stationery supplies. Client-server model, enables real-time data exchange and ensuring data integrity across the system.

Global Weather

I have developed a comprehensive weather application using HTML, CSS, and JavaScript, with real-time weather data sourced from the OpenWeatherMap API.
